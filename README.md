## Methodology

### Environmental Constraints (Primary Focus)

This project applies an environmental screening approach to identify areas that are suitable from an ecological and land-use perspective.

The following constraints were used:

- Protected areas (WDPA)
- Key Biodiversity Areas (KBA)
- Temple buffer zones (cultural protection)
- River buffer zones
- Coastal buffer zones

### Terrain Constraint

To ensure basic physical feasibility, slope limitations were applied:

- Solar PV: slope ≤ 5°
- Onshore Wind: slope ≤ 15°

> Note: This analysis focuses on environmental and terrain suitability only. 
> Technical and economic factors such as grid proximity, infrastructure access, wind resource quality, and solar irradiance were not included and can be considered in future work.
